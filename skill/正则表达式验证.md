# 正则表达式验证

> 2025年4月5日

验证身份证号

```regex
isIdentityNum = /^[1-8]{2}[0-9]{4}[0-9]{4}((0[1-9]{1})|(1[0-2]{1}))((0[1-9]{1})|(1[0-9]{1})|(2[0-9]{1})|(3[0-1]{1}))[0-9]{3}[0-9xX]{1}$/
```

验证QQ号

```regex
isQqNumberCheck = /^[1-9]{1}[0-9]{4,14}$/;
```

验证手机号

```regex
isPhoneNumCheck = /^[1]{1}[0-9]{10}$/;
```
