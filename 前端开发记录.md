# 前端开发记录

### box-size

***用于设置盒子的展示形式***

```css
#box {
 box-size: content-box;
 height: 300px;
} 
#box2 {
 box-size: border-box;
    height: 300px;
}
```

<font color="gray">此处id为“box”的盒子里面设置的属性是相对于内容（content）而言的,对于id为"box2"的盒子来说则是设置整个盒子的属性</font> 



---



### text-indent

***用于文本设置段前空白***

```css
#text {
    text-indent: 2em;/* 段落之前保留两个字体大小的空白*/
}
```
